# 지재권*지원과제 분석

## 역할 정의

IP·중기부 지원과제 데이터 분석가는 세부 전략분야 단위별 지재권 DB와 중기부 지원과제 분석 DB를 토대로 데이터 분석을 수행하고, 지재권과 지원과제 동향의 교차 분석을 통해 집중 영역과 공백 영역을 도출하여 R&D 필요 영역을 제시한다.

## 분석 범위

| 구분 | 분석 항목 | 작성 포인트 |
| --- | --- | --- |
| 지재권 DB | 출원인, IPC/CPC, 키워드, 패밀리, 인용, 권리상태 | 특허 집중도와 회피 가능성 |
| 지원과제 DB | 과제명, 수행기관, 지원부처, 기간, 예산, 성과 | 정부 투자 집중도 |
| 교차 분석 | 특허 밀집·과제 밀집, 특허 공백·과제 공백 | 신규 R&D 진입 영역 |
| 세부 전략분야 | 분야별 집중·공백 매트릭스 | 후보 품목 도출 근거 |

## GPT 검색 프롬프트 예시

- 첨단패키징 분야에서 특허는 집중되어 있으나 중기부 지원과제가 부족한 R&D 공백 영역을 도출하는 분석틀을 만들어줘.
- 지재권 DB와 지원과제 DB를 교차 분석해 집중·공백·추격·선점 영역으로 분류하는 표를 작성해줘.
