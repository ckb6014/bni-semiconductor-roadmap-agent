# 마크다운5. 전략품목 후보군 구성

## 역할 정의

전략품목 후보군 구성가는 정책·산업·시장·기술 환경분석 결과와 지재권·지원과제 동향 분석 결과를 종합하여 반도체 분야 세부 전략분야별 전략품목 후보군을 구성한다.

## 후보군 구성 기준

| 기준 | 판단 질문 | 배점 방향 |
| --- | --- | --- |
| 정책 부합성 | 2024-2026 정책·지원사업과 직접 연결되는가 | 정책 수혜 가능성이 높을수록 우선 |
| 시장성 | 수요처가 명확하고 수급 이슈가 존재하는가 | 판로와 성장성이 높을수록 우선 |
| 기술성 | 중소기업이 R&D로 차별화 가능한 기술 장벽이 있는가 | 기술 내재화 여지가 클수록 우선 |
| IP·과제 공백 | 특허·지원과제 분석에서 공백 또는 선점 가능 영역인가 | 공백·선점 영역일수록 우선 |
| 사업화 가능성 | 표준, 인증, 규제, 실증, 양산 전환 경로가 있는가 | 상용화 경로가 구체적일수록 우선 |

## GPT 검색 프롬프트 예시

- 환경분석과 지재권·지원과제 분석 결과를 종합해 반도체 중소기업 전략품목 후보군 10개를 우선순위 표로 제시해줘.
- 첨단패키징, 전력반도체, AI 반도체, 장비·소재 분야별 전략품목 후보를 도출해줘.
